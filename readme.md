## EE695: Net Zero Energy Building with IoT Integration

## Overview
This project designs and develops a solar powered Net Zero Energy Building (NZEB) prototype with IoT integration for real-time monitoring and control.

**Author:** Chukwudi Obianyo  
**Supervisor:** Bob Lawlor  
**Institution:** Maynooth University, Department of Electronic Engineering

## Repository Structure

## Key Results
- Annual energy saving: **23.7%** (1,892 kWh) via IoT optimization
- Grid import reduction: **41.5%**
- Payback period: **5.31 years** (with Irish SEAI grants)
- Working prototype with ESP32, Firebase and Flutter

## Requirements
- **Simulation:** MATLAB R2023b+, PVsyst 7.0+, DesignBuilder 6.0+
- **Firmware:** Arduino IDE 2.0+, ESP32 board package
- **Mobile App:** Flutter 3.0+, Dart 3.0+

## Hardware Used
- ESP32-WROOM-32 microcontroller
- 15W 5V monocrystalline solar panel
- MPPT P&O algorithm circuit
- INA219 current/voltage sensor
- DHT11 temperature/humidity sensor
